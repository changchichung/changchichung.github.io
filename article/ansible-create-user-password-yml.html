建立一個 user:passwd 格式的檔案

Created At：2022-10-27 14:12 Updated At：2022-10-27 14:13
Keywords: ansible yaml yml

htpasswd 要的帳號密碼對應
也可以拿來做其他用途

- name: make sure /tmp/user_pass exists
  file:
    path: /tmp/user_pass
    state: touch
  
- name: register previous mapping
  shell: "cat /tmp/user_pass"
  register: "user_pass_reg"
  
- name: create user/password mapping yml
  lineinfile: 
    line: "{{ item.name }}:{{ lookup('ansible.builtin.password', '/dev/null length=8 chars=ascii_letters,digits') }}"
    path: /tmp/user_pass
  with_items: "{{ users_nopassword }}"
  when: not user_pass_reg.stdout.find(item.name) != -1
