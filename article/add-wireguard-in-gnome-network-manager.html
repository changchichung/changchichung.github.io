add wireguard in gnome network-manager

Created At：2023-03-15 09:50 Updated At：2023-03-15 09:54

offcial site: https://github.com/max-moser/network-manager-wireguard

install packages

sudo apt install build-essential libgtk-3-dev libnma-dev libsecret-1-dev intltool

git clone repo and make

git clone https://github.com/max-moser/network-manager-wireguard
cd network-manager-wireguard
./autogen.sh --without-libnm-glib

./configure --prefix=/usr \
              --without-libnm-glib \
              --sysconfdir=/etc \
              --libdir=/usr/lib/x86_64-linux-gnu \
              --libexecdir=/usr/lib/NetworkManager \
              --localstatedir=/var
make
sudo make install

now open the gnome network manage panel and you can see the wireguard protocol !
